My Kobo H2O modifications:

- AutoPatch -

Perform patches directly on the Kobo. Beta stage. Unfinished.
http://www.mobileread.com/forums/showthread.php?t=253265

- AutoShelf -

Create collections for each folder directly on the Kobo. Beta stage.
http://www.mobileread.com/forums/showthread.php?t=254554

- HyphenDicts -

Move all hyphenation dictionaries to USB storage /.hyph/ for editing.
http://www.mobileread.com/forums/showthread.php?p=3004080#post3004080

- MagicMemory - 

Replace the Kobo's internal SD card on the fly.
http://www.mobileread.com/forums/showthread.php?t=261578

- ScreenSaver -

Display a random screensaver PNG image on standby/poweroff. H2O only.
http://www.mobileread.com/forums/showthread.php?t=255238

- TelnetFtp -

Enable telnet and ftp servers. Survives most firmware updates.

- ext234vfat -

Add support for ext2/3/4 filesystems for USB and SD storage.
http://www.mobileread.com/forums/showthread.php?p=3003110#post3003110
